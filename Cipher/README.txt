Modified Ceasar Cipher.

Главная задача - создать программу, шифрующую дешифрующую текст по шифру цезаря для двух языков одновременно (Русский, Английский).

Общепринятая реализация сводится к созданию массивов с буквами из словаря. При данной реализации нет возможности шифровать текст на двух и более языках. Так же, знаки пунктуации подвергаются удалению для удобства шифра.



Особенности моей реализации :

• Алгоритм работает без создания массива букв. Использована символьная таблица ASCII, работа с номерами символов из таблицы

• Учтены знаки пунктуации, они сохраняются в независимости от шифрования

• Нет необходимости дополнительно контролировать формат буквы (заглавная или прописная)
